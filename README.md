# Record-Based Medical Diagnostic Assistant

CMPE 255 Spring 2026 — San José State University.

A hybrid clinical decision-support pipeline. Mines symptom→disease
association rules from a patient transaction table and grounds each
prediction in retrieved biomedical literature.
